# 🌳 SCT_DS_03 - Decision Tree Classifier

## 📌 Project Overview

This project builds a **Decision Tree Classifier** to predict whether a customer will subscribe to a **bank term deposit** using the Bank Marketing dataset. The project includes data preprocessing, model training, evaluation, feature importance analysis, and decision tree visualization.

## 🎯 Objective

Develop a Decision Tree model that can classify customers based on their demographic and banking information and predict whether they will subscribe to a term deposit.

## 📂 Dataset

* **Dataset:** Bank Marketing Dataset (`bank.csv`)
* **Target Variable:** `y`
* **Records:** 4,521
* **Features:** Customer demographic and campaign-related information

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## 📋 Steps Performed

* Loaded the Bank Marketing dataset
* Checked dataset information and missing values
* Encoded categorical variables using Label Encoding
* Split the dataset into training and testing sets
* Built a Decision Tree Classifier
* Evaluated the model using Accuracy Score, Confusion Matrix, and Classification Report
* Visualized Feature Importance
* Generated and visualized the Decision Tree

## 📈 Output

* Accuracy Score
* Confusion Matrix
* Classification Report
* Feature Importance Chart
* Decision Tree Visualization

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install pandas matplotlib scikit-learn
```

2. Place `bank.csv` and `task03.py` in the same folder.

3. Run the program:

```bash
python task03.py
```
